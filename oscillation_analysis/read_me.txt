## Image analysis ##

1) run *<downsizeTimelapseImages.py> to get a scaled-down trans light movie for annotating ecdyses

2) create a txt file 'skin.txt' to annotate the ecdyses timepoints

3) run *<GUI_oscillation.py> to manually select the body axis, straighten the images and, for the wrt-2 analysis, manually label seam cells

4) run *<seamCell03_analysis_expression_1.py> to compute gene expression level in single seam cells

## plot results fopr mlt-10 ##

5) use *<moltCycle_results_kymograph.py> to create a kymograph of a single animal (figure 4 panel A)

6) use *<moltCycle_results_bodySections.py> to create figure 4 panel B

7) use *<moltCycle_results_timeseriesExpression.py> to create figure 4 panel C

8) use *<moltCycle_results_MaximaVSEcd.py> to create figure 4 panel D,E

9) use *<moltCycle_results_movieRawImages.py> to create a movie of single worm

## plot results fopr mlt-10 ##

10) use *<seamCell_results_timeseriesExpression.py> to create figure 4 panel G

11) use *<seamCell_results_MaximaVSEcd.py> to create figure 4 panel D,E
